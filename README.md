# SRE and Business Observability Platform

A production-like learning project combining:

- AWS
- Linux
- Docker
- Kubernetes and Amazon EKS
- CI/CD
- OpenTelemetry
- Prometheus, Grafana, Loki and Tempo
- SLI, SLO and Error Budgets
- Incident Management
- Customer Journey and Business Observability
- Open-source AIOps

## Target outcome

Build practical competencies required for a Middle SRE role, with selected engineering and architectural skills approaching Senior level.

Week 1, Day 1: AWS account foundation and local workstation setup.
